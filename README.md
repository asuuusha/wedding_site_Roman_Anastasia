# Wedding Site

## Структура файлов

```text
wedding-site/
├── index.html
├── style.css
├── script.js
└── assets/
    ├── images/
    │   ├── envelope-cover.png
    │   ├── torn-edge.png
    │   ├── hero.jpg
    │   ├── venue.jpg
    │   ├── details.jpg
    │   └── dresscode.jpg
    └── audio/
        └── wedding-song.mp3
```

## Что нужно загрузить вручную

- `assets/images/envelope-cover.png` — заставка-конверт
- `assets/images/torn-edge.png` — PNG рваного края
- `assets/images/hero.jpg` — первое фото
- `assets/images/venue.jpg` — фото места проведения
- `assets/images/details.jpg` — фото для блока «Пожелания»
- `assets/images/dresscode.jpg` — картинка дресс-кода
- `assets/audio/wedding-song.mp3` — музыка

В `index.html` замени `https://forms.google.com` на ссылку своей Google Form.
